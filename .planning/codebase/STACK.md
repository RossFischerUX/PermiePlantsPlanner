## Summary
Permaculture Plant Picker is a Next.js 14 (App Router) application written in TypeScript strict mode, styled with Tailwind CSS 3.4, and backed by Supabase (PostgreSQL + Auth + RLS). Build and tooling are handled by Next.js's built-in bundler (Turbopack-compatible), with tsx used for running data import scripts outside the main build. E2E testing only via Playwright targeting the production URL.
